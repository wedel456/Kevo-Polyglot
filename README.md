# kevo-polyglot
This is a Kevo Plus interface for ISY Polyglot.

MIT license.

#Requirements
`Kevo Gateway aka Kevo Plus`

Install:

1. Add your username and password to a new file called login.py
  * `echo USERNAME='"<username>"' >> login.py`
  * `echo PASSWORD='"<password>"' >> login.py`
2. Restart Polyglot
